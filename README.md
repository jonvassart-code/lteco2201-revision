# LTECO 2201 · Fiches de Révision

Interface interactive de révision pour le cours **Sociétés, Cultures, Religions : Lectures Bibliques** (UCLouvain, 2025-2026 Q2).

## Contenu

- **Fiches de révision** — 7 séances, points clés + grille d'analyse pour l'examen
- **30 Questions** — Auto-évaluation avec réponses cachées
- **Analyses de textes** — Les 9 textes majeurs avec points essentiels

## Déploiement sur Vercel

1. Push ce repo sur GitHub
2. Va sur [vercel.com](https://vercel.com), importe le repo
3. Vercel détecte Vite automatiquement → clique **Deploy**

## Dev local

```bash
npm install
npm run dev
```
